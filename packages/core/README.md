# @mcp-kit/core

Transport-independent core APIs for `mcp-kit`.

## Scope

`@mcp-kit/core` owns:

- app assembly through `createMcpApp()`
- public definitions for tools, resources, and prompts
- request context and policy contracts
- transport-independent middleware and runtime helpers

Concrete stdio and HTTP runtimes live outside this package.
